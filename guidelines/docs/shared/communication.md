# Communication

- If a file named `.dev-language` exists in the root of the project, use this language to communicate with the
  developer; except for code, commits, and branch names which must be in **English**.
- Else, default to **French**, except for code, commits, and branch names which must be in **English**.
- All code artifacts (variable names, function names, comments in code, commit messages, branch names) must be in
  English.
